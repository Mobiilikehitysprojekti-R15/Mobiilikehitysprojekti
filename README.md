# Skydive Mobile Application – Oulu University of Applied Sciences

This project is a mobile application developed as part of the Mobile Programming Application Project course (IN00ED17) in the Degree Programme in Information Technology at Oulu University of Applied Sciences.


## Goals

- To design, develp, and test a mobile application that runs on Android operating system
- To work as part of a development team using agile software development methods, and document the project work
- To design a user interface for the application
- To apply mathematics in various programming tasks

## Built with a modern, scalable mobile-first architecture using React Native and Firebase.

## Features
- **User accounts:** Secure registration and login
- **Profile:** Users can pick a profile picture and update their personal information
- **Weather:** Users can check the weather forecast for different dropzones and get visual indicatiors for skydiving-safe conditions
- **Log and statistics:** Users can log, browse and see history and statistics of their jumps
- **Light/Dark mode:** System-wide theme switching with persistent settings
